# Hackathon Resources & Platforms

Curated collection of platforms, GitHub repositories, and communities for hackathons across multiple domains — coding, AI, robotics, hardware, and general innovation.

---

## 1. Hackathon Platforms & Communities

- **Devpost** – https://devpost.com : Platform for finding and showcasing hackathons.  
- **Hackerearth** – https://www.hackerearth.com/challenges/ : Coding & innovation hackathons.  
- **MLH (Major League Hacking)** – https://mlh.io : Student-focused hackathon events.  
- **HackerNoon Hackathons** – https://hackathons.hackernoon.com : Tech-focused hackathons and contests.  
- **ChallengeRocket** – https://challengerocket.com : Global hackathons for developers.  
- **HackClub** – https://hackclub.com : Community for student hackers & hackathons.  
- **AngelHack** – https://angelhack.com : Global hackathon network.  
- **TechCrunch Hackathon Directory** – https://techcrunch.com/events/hackathon/ : Curated hackathon listings.

## 2. GitHub Repositories for Hackathon Projects & Resources

- **Awesome Hackathons** – https://github.com/aframevr/awesome-hackathons : Collection of hackathon resources & guides.  
- **MLH Hackathon Starter Projects** – https://github.com/MLH/mlh-hackathon-starter : Boilerplate projects for hackathons.  
- **Hackathon Ideas** – https://github.com/madhavanmalolan/hackathon-ideas : Curated list of hackathon project ideas.  
- **Awesome ML/AI Hackathon Projects** – https://github.com/search?q=hackathon+ai : Collection of AI & ML hackathon projects.  
- **Open Hackathon Projects** – https://github.com/topics/hackathon : Various open-source hackathon projects across domains.

## 3. Domain-Specific Hackathon Resources

### a. AI / Machine Learning
- **Kaggle Competitions** – https://www.kaggle.com/competitions : Data science and ML challenges.  
- **AI Hackathon Starter Kits** – https://github.com/topics/ai-hackathon : Open-source AI hackathon projects.  

### b. Robotics / Hardware
- **Hackster.io** – https://www.hackster.io : Hardware & IoT hackathon projects.  
- **Arduino Project Hub** – https://create.arduino.cc/projecthub : Hackathon-ready robotics/IoT projects.  
- **Raspberry Pi Projects** – https://projects.raspberrypi.org/en/hackathons : Ideas for hardware hackathons.  

### c. Web & App Development
- **Awesome Web Dev Projects** – https://github.com/sindresorhus/awesome : Includes web/app project ideas for hackathons.  
- **React Hackathon Starter** – https://github.com/Hashnode/react-hackathon-starter : Boilerplate for web hackathon projects.  
- **Flutter Hackathon Starter** – https://github.com/iampawan/FlutterExampleApps : Mobile app hackathon projects.  

### d. General Innovation / Interdisciplinary
- **Hackathon Resources Wiki** – https://github.com/hackathon-resources : Guides, checklists, and project templates.  
- **Project-Based Hackathons** – https://github.com/topics/project-hackathon : Repository collection across multiple fields.  
- **Open Source Hackathon Ideas** – https://github.com/topics/hackathon-ideas : Curated lists of project ideas.
